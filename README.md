# graphql-comment

Comment System based on graphql compose

## Installation

  ```
  $ npm install git+ssh://git@github.com:digithun/graphql-comment.git
  ```

## Development
1. **Run MongoDB:**

  ```
  $ docker run -d -p 27017:27017 mongo
  ```
2. **Run GraphQL server:**

  ```
  $ npm run dev
  ```
